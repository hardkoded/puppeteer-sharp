Dispose()

Releases all resource used by the Connection object. It will raise the Closed event and dispose Transport.

Declaration
public void Dispose()
Remarks

Call Dispose() when you are finished using the Connection. The Dispose() method leaves the Connection in an unusable state. After calling Dispose(), you must release all references to the Connection so the garbage collector can reclaim the memory that the Connection was occupying.
